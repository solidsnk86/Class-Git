Secuencias de Escape:
\' = Mostrar una comilla simple dentro de una cadena en la salida de información
\" = Mostrar una comilla simple dentro de una cadena en la salida de información
\\ = Barra invertida, diagonal inversa, es para mostrar un back slash
\n = Esto es un salto de línea
\r = Retorno
\t = Tabulación
\b = Retroceso
\f = Salto de páginia
\g = Escape global para expresiones regulares
